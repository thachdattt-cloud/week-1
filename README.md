\# Student Management - Week 1



Console app quản lý sinh viên bằng C#, bài tập tuần 1 môn Backend.



\## Nội dung tuần 1

Bài tập nhằm luyện: kiểu dữ liệu, câu điều kiện, vòng lặp, collection cơ bản (List, HashSet, Dictionary).



\## Các bài đã hoàn thành

\- \*\*Bai1 - Bai5\*\*: Console cơ bản, tính điểm, điều kiện, vòng lặp, quản lý danh sách sinh viên (List)

\-bai 6,7(dang update)

\## Công nghệ

\- C# / .NET

\- Console Application
